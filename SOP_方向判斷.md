# 🎯 方向判斷SOP — 先Scan方向，後Scan期權

> **教訓：唔好同時掃CALL+PUT。先判斷方向，再掃嗰一邊嘅期權。**
> 生成日期：2026-06-26

---

## 流程

```
Step 1: 方向判斷（開3-5個subagents同時驗證）
├─ Gann正方：現價 vs 關鍵線，方向偏上/下？
├─ Fib時間節點：而家喺邊個時間位？轉折未？
├─ Elliott Wave：而家咩浪？確認率幾多%？
├─ 鏈上數據：邊個接貨？邊個沽貨？大戶方向？
├─ 爆倉位地圖：邊個方向更易被逼倉？
├─ 資金費率：空頭crowded定多頭crowded？
└─ Order Book結構：近端Bid/Ask邊個薄？結算日效應？

Step 2: 方向確認 → 評分
├─ CALL > 60% → 只掃CALL
├─ PUT > 60% → 只掃PUT
└─ 50/50 → 雙開Strangle或等

Step 3: 根據方向掃Derive.xyz期權
├─ DTE同目標對齊
├─ 打和點合理（低過目標價）
├─ Spread/流動性合格
└─ 成本可控

Step 4: 入場
├─ 掛Limit，唔好市價追
├─ 分注食糊（對齊爆倉位）
└─ 止蝕鐵律
```

## 結算日特別規則

| 時期 | 莊家行為 | 方向影響 |
|------|---------|---------|
| 結算前24h | 壓價向Max Pain | 通常偏淡/震盪 |
| 結算後 | Gamma解鎖，方向自由 | 跟趨勢行 |

> 🧠 結算日前唔好重注賭方向。結算後Gamma解鎖先係真正方向。
